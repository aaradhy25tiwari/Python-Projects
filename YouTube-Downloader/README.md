# YouTube Video Downloader using Python 
## Requirements for this project are:
+ Python 3.10 and above
+ pytubefix

---

## Repository Structure

    YouTube-Downloader/
    ├── main.py
    ├── requirement.txt
    └── README.md

- **main.py** — the main file which downloades a YouTube video
- **README.md** — this documentation
- **requirements.txt** — List of modules required for this project

---

## Setup Instructions

1. **Clone** the repository:

        git clone https://github.com/aaradhy25tiwari/Python-Projects.git

        cd Python-Projects/YouTube-Downloader

2. Install dependencies:

       pip install -r requirements.txt

3. Run the web UI:

        cd YouTube-Downloader
        python app.py
        Open http://127.0.0.1:5000 in your browser
