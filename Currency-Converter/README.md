# Currency Conversion system using Python 
## Requirements for this project are:
+ Python 3.10 and above

---

## Repository Structure

    Currency-Converter/
    ├── main.py
    ├── requirement.txt
    └── README.md

- **main.py** — the main file where the conversion is implemented  
- **README.md** — this documentation
- **requirements.txt** — List of modules required for this project

---

## Setup Instructions

1. **Clone** the repository:

        git clone https://github.com/aaradhy25tiwari/Python-Projects.git

        cd Python-Projects/Currency-Converter

## Usage

1. Create a ".env" file
2. Get your API Key from [Free Currency API](https://freecurrencyapi.com/)
3. Store your API Key in the .env file in the format:
        
        API_KEY="your_api_key"

4. Run the code
